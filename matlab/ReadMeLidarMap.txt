Description;
The GUI has pushbuttons to move the robot in any direction, and one button to cease movement.
There is also one button to open up a serial port to connect with the bluetooth device which utilize UART.
Then there is one more button which triggers a LIDAR scan, the data is then displayed as a plot. 

The robot is a standalone application developed during the course TNE095 - Projektkurs - Elektronikdesign med projektledning.
This course has its main focus on the GUI fabrication in Matlab using the GUIDE program.
